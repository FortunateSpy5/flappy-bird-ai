# flappy-bird-ai
## Flappy Bird AI Using Genetic Algorithm and Pygame
### Controls
* F1: Start display rendering (deafult)
* F2: Stop display rendering (speeds up computation)
* 1: 1 FPS (frames per second)
* 2: 90 FPS
* 3: 180 FPS
* 4: 900 FPS
* 5: Unlimited FPS (default)
